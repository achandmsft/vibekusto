# Demo Visualize — vibekusto Showcase Dashboards

Three single-prompt dashboards generated by vibekusto using the **free public demo cluster** `https://help.kusto.windows.net`. Each dashboard was created from one natural-language prompt — no KQL written by hand. This project demonstrates **Visualize mode** — ask a question, get a dashboard.

| Dashboard | Live Preview | Prompt |
|---|---|---|
| US Storm Damage | [storm_dashboard.html](https://achandmsft.github.io/vibekusto/projects/demo-visualize/storm_dashboard.html) | Show me storm damage by state, event type, and month… |
| Contoso Sales | [sales_dashboard.html](https://achandmsft.github.io/vibekusto/projects/demo-visualize/sales_dashboard.html) | Build a sales dashboard from ContosoSales… |
| Storm Seasons | [seasons_dashboard.html](https://achandmsft.github.io/vibekusto/projects/demo-visualize/seasons_dashboard.html) | Analyze temporal storm patterns… |

## Re-generate

```bash
python run_demos.py
```

This runs against `https://help.kusto.windows.net` and recreates all 3 dashboards. Requires `az login` and `pip install azure-kusto-data azure-identity`.

## Delete when ready

This folder exists as a reference. When you're ready to use vibekusto for your own data, feel free to delete it:

```bash
rm -rf projects/demo-visualize
```

The vibekusto agent and all setup files live outside this folder and are unaffected.
